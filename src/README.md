# 📅 Interactive Calendar Component

A modern, responsive, and visually appealing **Interactive Calendar Web App** built using React. This project is inspired by a physical wall calendar design and includes advanced features like date range selection, notes integration, and smooth UI interactions.

## 🛠 Tech Stack

* ⚛️ React (Vite)
* 🎨 CSS (Custom Styling)
* 🎬 Framer Motion (Animations)
* 📦 LocalStorage (Data Persistence)

---

## ✨ Features

### 📌 Core Features

* 🗓 **Wall Calendar UI**

  * Inspired by real-world hanging calendars
  * Hero image section with month overlay

* 📅 **Dynamic Calendar**

  * Correct days & month handling
  * Weekday alignment (Sun–Sat)

* 🔄 **Month Navigation**

  * Switch between months seamlessly

* 🎯 **Date Range Selection**

  * Select start and end date
  * Highlight selected range visually

* 🧠 **Notes Integration**

  * Add notes for selected date range
  * Stored using LocalStorage

---

### 🎨 UI/UX Enhancements

* ✨ Clean & minimal design
* 🎯 Hover & click interactions
* 🌈 Gradient highlights for selected dates
* ⭐ Today indicator
* 📱 Fully responsive (Mobile + Desktop)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone   https://github.com/itsmetushar09/calender-app
```

### 2️⃣ Navigate to project

```bash
cd calendar-app
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the app

```bash
npm run dev
```
