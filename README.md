# 📅 Interactive Wall Calendar (React)

A modern, responsive, and visually polished **Interactive Calendar Component** built using React (Vite).
Inspired by a **physical wall calendar**, this project combines aesthetics with functionality — including date range selection, notes, animations, and dynamic theming.

## 🛠 Tech Stack

* ⚛️ React (Vite)
* 🎨 CSS (Custom + Glassmorphism UI)
* 🎬 Framer Motion (Animations)
* 📦 LocalStorage (Data persistence)
* 📅 JavaScript Date API

---

## ✨ Features

### 🧱 Core Functionality

* 📅 **Dynamic Calendar Grid**

  * Correct days & month handling
  * Proper weekday alignment

* 🔄 **Month Navigation**

  * Smooth switching between months

* 🎯 **Date Range Selection**

  * Select start and end dates
  * Intelligent range handling (auto-correct if reversed)

* 🧠 **Notes System**

  * Add notes for selected date range
  * Persisted using LocalStorage

---

### 🎨 UI & UX (Wall Calendar Aesthetic)

* 🖼 **Hero Image Section**

  * Inspired by real wall calendars
  * Month + year overlay on image

* 🎬 **Flip Animation**

  * Smooth page-like transition when changing months

* 🌈 **Dynamic Theme**

  * Color theme changes based on month

* ⭐ **Today Indicator**

  * Highlights current date accurately

* ✨ **Micro Interactions**

  * Hover scaling
  * Click feedback
  * Smooth transitions

---

### 📱 Responsiveness

* 💻 Desktop: Split layout (image + calendar)
* 📱 Mobile: Stacked layout (fully usable)
